
Mini Library Management System (Python)
Files included:
- library.py       : Core functions and data structures (lists, tuples, dictionaries)
- main.py          : Simple CLI to interact with the system
- tests.py         : Automated tests / usage examples (run to check functionality)

How to run:
1) python tests.py   -> runs automated tests
2) python main.py    -> run a simple interactive CLI

Notes:
- books are stored as dictionaries inside a list (books).
- genres are stored in a tuple to show an immutable collection.
- borrow_records uses a list of tuples (member_id, isbn).
